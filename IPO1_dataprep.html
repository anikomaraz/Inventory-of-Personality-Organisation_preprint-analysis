Missing data

Lets have a look at missing data structure of IPO to (hopefully) save some data by imputing.

library(VIM)

87.7% of data are not missing any value (see plot on the right hand side), and 3.2% (so 6 participants) miss data on all IPO items. These will have to be excluded from the final database. Other than this the distribution of missing data seems random to me. In terms of missing data of the items, Items 33 and 24 miss most data, although only 4.8% which isnt very much. So we can proceed with data imputation.

Data imputation

# So we want to get rid of rows which contain no data.
IPO_items_noBlankRow <- df_patient[grep("^SZSZK", names(df_patient), value=T)][!(rowSums(is.na(IPO_items)) == "83"),] 

# The rest can be inputed:
dim(IPO_items_noBlankRow) 
[1] 181  83
# let's do the imputation 
library(mice)
IPO_items_imputed <- mice(IPO_items_noBlankRow, m=2, maxit=50, method="pmm", seed=1239)
# and pick the 2nd imputed matrix as the final dataset which hopefully contains no missing data
IPO_items_complete <- complete(IPO_items_imputed, 2) 
dim(IPO_items_complete)
## [1] 181  83
